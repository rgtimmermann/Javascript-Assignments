How does CodePen/JSFiddle/JS Bin work?

En hjememside hvor du kan finde koder, kopier dem og bruge dem i dit eget projekt. Du kan også finde koder derinde og ændre koden derinde med det samme og se hvad der sker, det du selv skriver forsvinder dog hvis/når du opdaterer siden.

Is there a live output? (Can you see the output of the code you have made?)

Ja, så snart du ændre noget kan du se ændringen med det samme. Du kan altid følge hvad du laver og se ændringerne med det samme.
