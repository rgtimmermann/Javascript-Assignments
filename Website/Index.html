Guide til brugen af Git og GitHub

Første skridt er at installere Git på din PC

Git kan downloades her


Dernæst kan jeg anbefale at følge RIkkes trin på hendes flotte hjemmeside her

Hvis du vil tilføje en eller flere filer i din mappe, skal du ligge dem derind og derefter hoppe over i terminalen

1. Skriv: git status

Du får skrevet noget lignende: On branch main Your branch is up to date with 'origin/main'. Untracked files: (use "git add ..." to include in what will be committed) daily_life_w1.jpeg

Her vil jeg adde et billede "daily_life_w1.jpeg". Læg mærke til, at filen er rød (endnu ikke pushed)


2. Skriv nu: git add .

Her mødes du ikke af nogen besked - du har altså gjort det rigtigt


3. Skriv: git commit -m "din commit message"

Jeg skrev for eksempel: git commit -m "added an image"

beskeden lyder nu: [main 3463c9d] added an image 1 file changed, 0 insertions(+), 0 deletions(-) create mode 100644 daily_life_w1.jpeg


4. Skriv nu: git push

Herefter får du en lang besked lignende: Enumerating objects: 4, done. Counting objects: 100% (4/4), done. Delta compression using up to 8 threads Compressing objects: 100% (3/3), done. Writing objects: 100% (3/3), 85.77 KiB | 17.15 MiB/s, done. Total 3 (delta 0), reused 0 (delta 0), pack-reused 0 To https://github.com/rgtimmermann/Javascript-Assignments.git 05152a1..3463c9d main -> main

Den giver dig tilmed et link til din GitHub
